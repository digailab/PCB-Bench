PCB-Bench

PCB-Bench: Benchmarking LLMs for Printed Circuit Board Placement and Routing (ICLR 2026)

📄 Paper (OpenReview) 💻 Code (GitHub) 🌐 Project Page

PCB-Bench is a comprehensive benchmark designed to systematically evaluate (multimodal) large language models (LLMs/MLLMs) for PCB placement and routing by integrating text, images, and real PCB design artifacts into a unified evaluation framework.

What's Included

~3,700 Text QA/CQ ~500 Multimodal Problems 174 Real-world PCB Projects
